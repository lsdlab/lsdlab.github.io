可算是稍稍安定了

Posted by Chen Jian on Sat 19 September 2015 Updated on Sat 19 September 2015

被虐结束

面试 Python 岗位受挫，一直被虐，我还需要一些时间继续研究 Python，造 Python 轮子，丰富 GitHub 和简历。

开始实习

前端岗位，小公司，有自己的渠道取得业务，中华门，二十公里，能够骑车去，下雨的话就地铁。应该不会非常难，时间上也会比较空闲，有些时间搞自己的事情。前端也不错，还是和数据相关，实习薪酬三千，第一个月试用期两千五，六个月合同，将根据表现浮动。十一之后入职，我想后面将会比较充实，工作，看书，骑车（美利达 R903 已准备入手），玩游戏（需要买个 Xbox 360，已经加入购物车），可以开张单子，把南京能吃的特色的小馆子，能接受的小吃，全吃一遍，到处骑车玩玩。

决定

还是需要时间，继续学习基础知识，还有 Python，希望能够在毕业的时候找一个正式工作，做基础设施或者数据相关的 Python 岗位。关于找工作，和毕业后在哪里发展，我现在不再特别要求一定要去哪里，反正以现在的能力去哪里都是一样的，还是得多学习，积累真正的实践经验。正式工作将不再“瞎”投简历，一定投业务成熟，且我要对业务感兴趣。这个时候就显得技术型公司或者技术占主导的公司的优势了，不需要像是现在的各种小型初创公司，初期不仅技术粗糙，还要为业务扯皮，加班赶进度，根本没法搞。

就这么定了。
