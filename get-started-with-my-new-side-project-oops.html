Get started with my new side project: oops

Posted by Chen Jian on Thu 08 December 2016 Updated on Thu 08 December 2016

我想做一个自用的个人运维平台，就是把 VPS 的简单操作和监控都放在一个面板上面，还有博客的访问统计，简单的进程监控，还有可以用 Telegram 的 Bot 做些简单操作，图床工具和短链接工具都能做进去，这个就用 API 特别多的 Python 来做了，原来是想用 Rails 做的，试了试，除了用户登录直接用脚手架就行了，其他方面都不适合，况且我这种应用也没有大量的 CRUD 操作，主要还是集成 API。

这些 side project 都要慢慢做了，现在我每天要花两个小时去健身，还请了私教，要开始多吃多练了。
