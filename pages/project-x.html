Spendingviz

GitHub: lsdlab/spendingviz

可视化我的日常支出数据，尝试产品化，搭建具有数据分析功能的后端平台，以及 iOS App。

终于有个自我感觉稍微能拿得出手的项目了，虽然是个很小的项目，但也想先做着自己用，满足自己的需求，然后尝试进行产品化。

代码里面很多地方都写死了，也不太想搞了，反正还要换 Django 做更加完善的应用以配合 iOS App。

Hackathon Starter CN

GitHub: lsdlab/hackathon-starter-cn

Deprecated

Node.JS + Express + MongoDB 做的脚手架，基础的用户模块，登录注册、重置密码、Postmark 第三方邮件服务。

在爆发 MongoDB 勒索事件前我就换成的 PostgresSQL，真是机智……

很多 Web 框架要么就是自带用户模块，比如 Django，要么就是有个用的非常广泛的库，比如 Rails 的 deivse，但都非常不好用，Django的设计上就有问题，Rails 的 devise 太复杂，据说源码已经是到了乱套了的地步，不仅实现的复杂度太高，使用起来的复杂度也太高，我就想自己做个，然后留着给我的其他产品用，直接调用 API 作为用户管理的功能，这就需要 API 的性能要够高，Node.JS 正合适。

之前是废弃了，因为没用到，现在想想估计得有地方要用了，有时间的话要把 API 补上，或许考虑换掉 Express。
