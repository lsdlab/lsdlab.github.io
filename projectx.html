[GitHub: lsdlab/hackathon-starter-cn][4]

Deprecated

Node.js + Express + MongoDB 做的脚手架，基础的用户模块，登录注册、重置密码、Postmark 第三方邮件服务。

在爆发 MongoDB 勒索事件前我就换成的 PostgresSQL，真是机智……

很多 Web 框架要么就是自带用户模块，比如 Django，要么就是有个用的非常广泛的库，比如 Rails 的 deivse，但都非常不好用，Rails 的 devise 太复杂，据说源码已经是到了乱套了的地步，不仅实现的复杂度太高，使用起来的复杂度也太高，我就想自己做个，然后留着给我的其他产品用，直接调用 API 作为用户管理的功能，这就需要 API 的性能要够高，Node.js 正合适。

之前是废弃了，因为没用到，现在想想估计得有地方要用了，有时间的话要把 API 补上，或许考虑换掉 Express。
